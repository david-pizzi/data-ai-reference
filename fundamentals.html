Fabric Architecture
Study: Fabric Concepts →
Microsoft Fabric is a unified SaaS analytics platform. All workloads share OneLake storage, a single security model, and one billing system — no infrastructure to manage.
Data Lifecycle
Study: Data Lifecycle →
Data flows through six stages in Fabric: Get data from sources, Store it in OneLake, Prepare and transform, Analyze with engines, Visualize in reports, and Integrate insights back into business processes.
Workspaces & Governance
Study: Workspaces →
Fabric organizes everything into a strict hierarchy: Tenant at the top, Capacities for billing, Workspaces for collaboration, and Items as the individual artifacts. Roles control access at each level.
Direct Lake Mode
Study: Direct Lake →
Direct Lake is a new storage mode that combines the speed of Import with the freshness of DirectQuery. It reads Delta-Parquet files directly from OneLake — no data copying, no query delegation.
